# About Me Project

## Start Dev Server
1. Install dependencies:
	```sh
	npm i
	```
2. Copy `.env.example` to `.env` in the project root:
	```sh
	cp .env.example .env
	```
3. Fill out all required environment variables in `.env`.
4. Start the development server:
	```sh
	npm run dev
	```

## Start in Docker
```sh
docker compose up
```
